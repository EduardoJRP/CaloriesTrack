## Installation

1. Clone the repository:

```bash
git clone https://github.com/EduardoJRP/CaloriesTrack.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open the app in your browser at https://localhost:3000
